# MEMORY_CORRECTIONS

## Calibration
- Do not assume Expo Go runtime failures are app-code regressions on Windows; path length and local watcher behavior can be the root cause.
- Prefer the shortest workspace path available before changing project code for native Android build issues on this machine.
- Do not treat the green paper tags as raw arXiv query terms; they are inferred topic labels applied after fetch and ranking.
- If the Brief screen does not surface the active topic subset explicitly, users will reasonably misread it as a generic popularity feed.
- Do not overstate the search path as semantic search; it is still keyword-style and limited by the backend query builder.
- Do not describe notifications as shipped end to end; only preference and token registration are implemented.
- The emulator text-entry overlay issue was an input-method quirk, not an app feature defect.
